About this Project

The Banking App Analyzer is a tool for scraping, analyzing, and visualizing Google Play Store data for banking applications. It provides insights into app ratings, reviews, and user sentiment.

Features

  • Google Play Store data scraping for banking apps
  • Review collection and analysis
  • Sentiment analysis of user reviews
  • Data visualization with charts and graphs
  • Interactive data tables

Technologies Used

  • Backend: Python with Flask
  • Scraping: google-play-scraper
  • Sentiment Analysis: TextBlob
  • Data Processing: Pandas
  • Frontend: HTML, CSS, JavaScript, Bootstrap 5
  • Visualization: Chart.js, DataTables.js

How It Works

1. Data Collection

Using google-play-scraper to fetch app information and reviews from the Google Play Store.

2. Processing & Analysis

Processing the data and performing sentiment analysis on user reviews.

3. Visualization

Presenting the data through interactive tables, charts, and dashboards.
